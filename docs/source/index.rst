Welcome to clashroyale's Documentation
======================================

The clashroyale module is an easy to use, feature-rich and async ready API wrapper for cr-api.com

**Features:**

- Use the same client for sync/async operations.
- Implements the entire cr-api API.
- Easy to use with an object oriented design.
- Optional caching for reliability and performance.
